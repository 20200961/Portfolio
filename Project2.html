담당 역할

AI 개발 - 수어인식모델 개발 및 프론트엔드 연결

프로젝트 소개

손수잇다는 AI 기술을 활용하여 누구나 쉽게 수어를 배울 수 있는 온라인 교육 플랫폼입니다.


실시간 손동작 인식 기술을 통해 학습자가 올바른 수어 동작을 하고 있는지 즉각적으로 피드백을 제공하며, 단계별 학습 커리큘럼을 통해 체계적인 학습이 가능합니다.


청각 장애인과 비장애인 간의 소통 장벽을 낮추고, 수어에 대한 접근성을 높이는 것을 목표로 개발되었습니다.

주요 기능

1
AI 기반 실시간 손동작 인식 및 정확도 분석
2
단계별 수어 학습 커리큘럼 및 진도 관리
3
학습 성취도 통계 및 개인별 맞춤 학습 추천
4
수어 사전 기능 - 단어 및 문장 검색
5
웹과 모바일 앱 기반의 통합 플랫폼

AI개발 및 유지보수 담당

- Python 기반 LSTM 딥러닝 모델을 활용한 수어 인식 기능 설계 및 구현


- 수어 영상 데이터 전처리/학습/검증 전 과정을 담당하여 모델 성능 안정화


- 학습된 수어 인식 모델을 배움터 학습 콘텐츠 및 수어 스피드 게임에 적용


- Flask 기반 API 서버 구축으로 AI 모델과 프론트엔드 연동 및 실서비스 적용


- AI 기능을 실제 교육 서비스에 통합하여 사용자 학습 경험 향상에 기여

기술 스택

Python TensorFlow React Native Flask MySQL AWS MediaPipe Node.js
