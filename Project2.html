프로젝트 소개

하이 한국은 한국을 방문하는 외국인 여행객들을 위한 맞춤형 여행지 추천 앱입니다.


사용자의 관심사와 위치 정보를 기반으로 개인화된 여행지를 추천하며, 실시간 번역, 교통편 안내, 주변 맛집 정보 등 여행에 필요한 모든 정보를 제공합니다.


다국어 지원과 직관적인 UI로 언어 장벽 없이 누구나 쉽게 사용할 수 있습니다.

주요 기능

1
AI 기반 개인 맞춤형 여행지 추천 알고리즘
2
실시간 다국어 번역 및 음성 인식 기능
3
GPS 기반 주변 관광지, 맛집, 숙소 정보 제공
4
여행 일정 플래너 및 경로 최적화
5
사용자 리뷰 및 평점 시스템
6
오프라인 지도 및 즐겨찾기 기능

기술 스택

React Native Node.js Express MySQL Google Maps API AWS
